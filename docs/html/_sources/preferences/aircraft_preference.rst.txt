Aircraft Preferences
====================

Use this panel to set add, edit or remove aircraft from the planner.

The aircraft characteristics are used to set the altitude and distances to the Top of Climb and
Beginning of Descent points.
 |aircraft preferences screenshot|

You can set the various speeds and rates of climb and descent as shown below.
